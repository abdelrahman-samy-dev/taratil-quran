# الأيقونات

يحتاج التطبيق إلى أيقونات PNG بحجمين:

1. `icon-192.png` - 192x192 pixels
2. `icon-512.png` - 512x512 pixels

## كيفية إنشاء الأيقونات:

### الطريقة 1: استخدام أداة online
- استخدم [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
- أو [RealFaviconGenerator](https://realfavicongenerator.net/)

### الطريقة 2: استخدام برنامج تصميم
- استخدم أي برنامج تصميم (Photoshop, GIMP, Canva)
- أنشئ أيقونة بسيطة تمثل القرآن الكريم (مثلاً: مصحف، آية، أو رمز إسلامي)
- احفظها بالحجمين المطلوبين

### الطريقة 3: استخدام SVG وتحويله
- أنشئ ملف SVG بسيط
- استخدم أداة تحويل SVG إلى PNG
- احفظ بالحجمين المطلوبين

## ملاحظات:

- استخدم ألوان متناسقة مع التطبيق (أخضر #2d5016)
- تأكد من أن الأيقونات واضحة حتى في الحجم الصغير
- الأيقونات يجب أن تكون مربعة (1:1 aspect ratio)
